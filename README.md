# Chinese Checkers

Virudhunagar style Chinese Checkers board game for 2-6 players on a single device.

**[Play now](https://shrihari1999.github.io/chinese-checkers/)**

## Features

- 2, 3, 4, or 6 player support (human only, turn-based)
- Standard hops and long hops (symmetrical gap jumping)
- Chain hopping within a single turn
- Installable as a PWA
