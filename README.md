# Chinese Checkers

Virudhunagar style Chinese Checkers board game for 2-6 players.

**[Play now](https://shrihari1999.github.io/chinese-checkers/)**

## Features

- 2, 3, 4, 5, or 6 player support (human only, turn-based)
- **Local mode**: pass-and-play on a single device
- **Online mode**: create a room and share the code with friends (realtime messaging via Ably)
- Standard hops and long hops (symmetrical gap jumping)
- Chain hopping within a single turn
- Undo moves within a turn
- Installable as a PWA
